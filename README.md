# 스프링 시큐리티와 OAuth2를 이용한 카카오 로그인
## 개발기간 
- 24.03.09 ~ 24.03.10

## 개발 환경
- java17
- **IDE** : intelliJ
- **Framework** : Springboot(3.x)
- **Database** : Mysql(8.1.0)
- **ORM** : JPA


## 주요기능
#### 카카오톡 로그인
- 로그인 시 카카오톡으로 로그인을 하고 스프링 시큐리티를 통하여 검증
